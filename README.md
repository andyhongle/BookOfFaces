# README

# BookOfFaces

BookOfFaces is a [Facebook](https://www.facebook.com/) clone, that allows users to create a profile and make text/image posts to interact with other users.

[Live link to project](https://bookoffaces.herokuapp.com/#/)

## Technologies Used:

* Ruby on Rails
* PostGreSQL
* JBuilder
* jQuery
* React
* Redux
* HTML
* CSS
